Technical Blog 5 - Problem Solving

Problem solving

As a programmer I face several code issues some of which are simple and some complex. I particularly had to execute media query which basically means creating my website responsive to different screen sizes starting from mobile first and then gradually progressing to bigger screen sizes. I specifically had to dig through the entire working of a media query and check for codes that are universal for various phones, phablets and desktop. First step was to use developer tools to check responsiveness on various screen sizes. Then modify the code step by step and check for correctness. I had to use online forum like stack overflow, raise my query and wait for an appropriate response. Then try to run codes and repeat the process for compatibility. Many a times it can get frustrating as you may come across several dead ends. The trick is look at each line of code and check for correctness. Run the codes and check if it is executing the expected way.

Using the JS Array docs as a starting point, describe what these functions do in your own words:

.map()

This method creates a new array with the results of the calling function on every targeted element on the array. For example, if we need the square root of all elements in an array we could use map to target elements in an array and assign the result in a new variable.

var numbers = [4, 9, 16];

var roots = numbers.map(Math.sqrt);

// roots is now [2, 3, 4], numbers is still [4, 9, 16]

.filter():

The filter() method creates a new array with all elements that pass the test implemented by the provided function. This method is quite handy to create an array out of an existing one with the required values. For Example:

var numbers = [12, 35, 46, 77, 89, 104, 39];

function isBigEnough(value) {return value % 2===0;}

var filtered = numbers.filter(isBigEnough);

confirm(filtered);

// filters all even numbers within the array

.reduce():

A method that takes a given array and reduces it to produce one result. For example, we can have an array with four elements and we want to add one with another.
