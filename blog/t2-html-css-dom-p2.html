What is grid based design?

A Grid based design is a system in which designers structure their layout on a series of horizontal and vertical intersecting grid , thereby structuring and organizing content in a presentable way. A grid is split in rows and columns and html elements are placed to provide the most user friendly browsing experience.

Importance of Responsive Webpages

Responsive web designs is an approach to code the website to enhance optimal viewing experience across various browsing devices. It's hyped because of increasing usage of web browsing across smartphones and tablets varying in screen resolution. Responsive designs as the name suggests "responds" to resizing or scrolling thereby facilitating ease of browsing.

What is semantic structure?

Sematic structure in html are tags that provide clear understanding to developers and browsers. Semantic tags include header 1-4 tags, Image and form.
